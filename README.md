# OneShot Game

Репозиторий сайта **oneshotgame.shop**.

## Разработка

```bash
npm install
npm run build
npm run server
```

Или с hot-reload фронтенда: `npm run dev:full`

## Docker (Linux)

```bash
cp .env.example .env
# Заполните .env
docker compose up -d --build
```

Приложение: `http://localhost:3001`

## Переменные окружения

См. `.env.example`.
